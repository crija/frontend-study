Trabalhando com listas

Listas ordenadas

Para fazer uma lista ordenada com numeração correta em HTML, utiliza-se a tag (ol) e em cada tarefa que for definir utiliza-se a tag (li). Podendo trocar o tipo do 'indicador' de tarefa usando o comando (type).

  1. Acordar
  2. Escovar os dentes
  3. tomar café
  4. Escovar os dentes
  5. Organizar agenda

Listas não ordenadas

As listas não ordenada são listas sem numeração. Para criar uma lista não ordenada, utiliza-se a tag (ul) e para cada tarefa, utiliza-se a tag (li).

  • Acordar
  • Academia
  • Estudar
  • Almoçar
  • Estudar
  • Tomar café da tarde
  • Correr
